# Genshin Teammate Finder — Telegram bot

Простой бот для поиска тиммейтов по Genshin: пользователи выбирают сервер, заполняют анкету, затем можно просматривать анкеты других.

Требования:
- Python 3.8+
- Telegram bot token (создать у @BotFather)

Установка и запуск:
1. Поместите все файлы (bot.py, db.py, requirements.txt, .env) в одну папку.
2. Установите зависимости:
   pip install -r requirements.txt
3. В файле `.env` уже должен быть ваш токен. (Если хотите, можно вместо .env установить переменную окружения TELEGRAM_TOKEN)
4. Запустите бота:
   python bot.py

Команды:
- /start — начать создание анкеты
- /browse — просматривать анкеты (после выбора сервера)
- /myprofile — посмотреть/удалить свою анкету
- /cancel — отменить текущую операцию

Замечания по безопасности:
- Токен в `.env` — удобно для локального теста. Не публикуйте `.env` в публичных репозиториях.
- Если токен стал публичным — сгенерируйте новый у @BotFather и замените в `.env`.

Если хотите, могу:
- Переписать на aiogram v3.
- Сделать Dockerfile для запуска.
- Добавить фильтрацию и кнопку "Написать автору" (потребует, чтобы пользователи начали диалог с ботом).
